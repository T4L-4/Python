# Operatörler

```python
Aritmetik operatörler:
(+, -, *, /, %, **, //)

Atama operatörleri:
(=, +=, -=, &=, %= ...)

Karşılaştırma operatörleri:
(==, !=, >, <, <=, >=)

Mantıksal operatörler:
(and, or, not)

Referans kontrol operatörleri:
(is, is not)

Üyelik operatörleri:
(in, not in)

Bitsel operatörler:
(&, |, ^, ~, <<, >>)
```
